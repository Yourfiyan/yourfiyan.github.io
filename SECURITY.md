# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.** Instead, email:

- **Email:** yourfiyan@proton.me

### Response time

You can expect an initial response within **48 hours**.

## Scope

This is a static portfolio website deployed on GitHub Pages. Security considerations include:

- Cross-site scripting (XSS) in user-facing content
- Third-party dependency vulnerabilities
- Open Graph and meta tag integrity
