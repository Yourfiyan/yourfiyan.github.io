# Contributing to yourfiyan.github.io

Thanks for your interest in contributing!

## How to Contribute

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/<your-username>/yourfiyan.github.io.git
   cd yourfiyan.github.io
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start development server:**
   ```bash
   npm run dev
   ```
5. **Create a branch**, make changes, and open a Pull Request

## Guidelines

- Follow existing code style (TypeScript, React functional components)
- Test changes in dev mode before submitting
- Don't commit build artifacts or node_modules
- Update documentation if needed

## Bug Reports

Open an issue with steps to reproduce, expected behavior, and actual behavior.
